## WeChat 文章学习：Superpowers + OpenSpec 协同开发

**来源**：https://mp.weixin.qq.com/s/1Ui39M4kyyz2-gSlD7OBdg  
**文章标题**：从古法编程到 AI 编程：Superpowers + OpenSpec 协同开发实战指南  
**发布时间**：2026-04-18（AI 编程最佳实战系列第 18 篇）

### 核心概念

- **OpenSpec**：需求层规范驱动，核心理念是"在编写任何代码之前，就对要做什么达成一致"。双文件夹模型：`specs/`（事实来源）+ `changes/`（变更提案）
- **Superpowers**：执行层 AI 编程工具，更稳而非更强，专注代码风格一致性
- **每份变更包含三个文件**：`proposal.md`（为什么做）+ `design.md`（技术方案）+ `tasks.md`（实施清单）
- **效果数据**：Token 消耗降低 30%-50%，返工率下降 60%+（来自 CSDN 对比测试）

### 适用场景

维护 3 年以上老项目时 AI 辅助编码遇到的：风格不一致、老功能改崩、接口不同步等问题。

## Playwright 抓取微信文章技巧

微信文章（mp.weixin.qq.com）使用 JS 动态渲染，`web_fetch` 和普通 HTTP 请求均无法获取完整内容。

**有效方案**：使用 Playwright 渲染 + `page.inner_text()` 或 `page.locator('#js_content').inner_text()`

```python
import asyncio
from playwright.async_api import async_playwright

async def fetch_wechat(url):
    async with async_playwright() as p:
        browser = await p.chromium.launch()
        page = await browser.new_page()
        await page.goto(url, wait_until="networkidle")
        # 方法1：直接全文
        text = await page.inner_text("body")
        # 方法2：抓取正文区
        text = await page.locator("#js_content").inner_text()
        await browser.close()
        return text
```
