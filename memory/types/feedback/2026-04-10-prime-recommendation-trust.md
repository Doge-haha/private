## 规则
当主脑给出充分的技术分析和理由时，用户会信任并采纳建议。

**Why:** 主脑提供了详细的对比分析（Claude Code vs OpenClaw vs lossless-claw），包括能力差距表格和实施路线图，让用户能够做出明智决策。

**How to apply:**
- 给出建议时，提供充分的分析和对比
- 列出优缺点和风险
- 用户会基于理性分析采纳建议
