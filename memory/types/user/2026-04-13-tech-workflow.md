## 开发环境
- 偏好简洁、可移植的脚本（Python/Bash）
- 代码需高健壮性+详细注释

## AI 工具使用
- 鼓励深度搜索（Deep Research）+ 多工具协同

## 文件管理
- 生成文件需命名规范，存放在 workspace 目录下
- 涉及代码变动优先考虑 Git 规范

## 安全禁忌
- 严禁未经授权上传本地配置或私密数据到第三方平台
